# ClassIsland：让教室大屏变成智能课表

ClassIsland 是一款适用于班级多媒体屏幕的开源课表信息显示工具，灵感源于 iOS 的“灵动岛”功能。

## 能做什么？

| 功能 | 说明 |
|------|------|
| 📅 课表显示 | 显示当天课表、当前课程、下一节预告 |
| ⏰ 自动提醒 | 上下课自动弹窗+音效+语音 |
| 🌤️ 天气信息 | 实时显示天气、温度、预警 |
| 📝 鼠标穿透 | 不影响正常授课，不占屏幕空间 |
| 🔧 课表编辑 | 支持 Excel 导入、临时换课、多周轮换 |

## 为什么推荐？

**替代黑板课表**
调课直接在电脑改，不用擦掉重写。

**自动提醒**
老师不用再看手表，到点自动提醒。

**附加信息**
同时显示天气、倒计日等，让屏幕“活”起来。

**开源免费**
没有任何收费，所有人都能用。

## 安装教程

1. 访问官网：classisland.tech
2. 下载最新版本
3. 安装 .NET 8.0 运行时
4. 运行安装程序
5. 导入课表（支持 Excel）
